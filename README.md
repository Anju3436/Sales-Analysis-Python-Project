# 📊 Sales Data Analysis Project


## 🌐 Live Project

## 📌 Overview
This project focuses on analyzing a sales dataset to uncover meaningful insights about business performance. Using Python and data visualization techniques, the project highlights trends, patterns, and key metrics.

---

## 🎯 Objectives
- Analyze overall sales performance  
- Calculate key metrics (total, average, max, min sales)  
- Identify trends and patterns  
- Create visualizations  
- Generate business insights  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

---

## 📂 Dataset
The dataset includes:
- Order details  
- Sales amount  
- Products  
- Regions/Cities  

---

## 📈 Key Metrics
- Total Sales: 506,562.52  
- Average Sales: 3446.00  
- Maximum Sale: 9774.03  
- Minimum Sale: 652.35  

---

## 📊 Visualizations
- Bar Chart (Sales comparison)  
- Pie Chart (Distribution)  
- Trend Analysis  

---

## 💡 Insights
- High-performing products generate more revenue  
- Sales vary across regions  
- Certain time periods show higher sales  
- Visualizations help understand patterns quickly  

---

## ✅ Conclusion
This project shows how data analysis can be used to extract insights and support better business decisions.
