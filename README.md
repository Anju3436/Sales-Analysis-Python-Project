# Sales-Analysis-Python-Project
A data analysis project using Pandas and Matplotlib to explore sales performance and business insights.
📊 Sales Data Analysis Project
📌 Overview
This project analyzes a sales dataset to understand business performance, revenue trends, and customer insights using Python.
🎯 Objectives
Calculate total, average, max, and minimum sales
Identify sales trends
Visualize data using charts
Generate business insights
🛠️ Tools & Technologies
Python
Pandas
Matplotlib / Seaborn
Jupyter Notebook
📂 Dataset
The dataset contains sales records including:
Order details
Revenue
Products
Cities
📈 Key Analysis
Total Sales: 506,562.52
Average Sales: 3446.00
Maximum Sale: 9774.03
Minimum Sale: 652.35
📊 Visualizations
Bar Chart (Sales comparison)
Pie Chart (Distribution)
Trend Analysis
💡 Insights
Some products generate significantly higher revenue
Sales vary across regions/cities
Monthly trends show growth patterns
✅ Conclusion
This analysis helps understand sales performance and supports better business decision-making using data.
